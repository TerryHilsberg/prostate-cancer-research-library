# Patient-Perspective Overview

This section turns the evidence library into a patient-oriented decision map.

## Typical journey

1. **Screening or suspicion** — PSA, DRE, family history, urinary symptoms, incidental finding
2. **Risk clarification** — repeat PSA, PSA density, MRI, risk calculators
3. **Diagnosis** — biopsy approach, pathology, Grade Group/Gleason, tumor volume
4. **Staging and aggressiveness** — imaging, clinical stage, genomic classifiers if appropriate
5. **Management choice** — active surveillance, surgery, radiation, systemic therapy, clinical trial, or palliative focus depending on risk/stage
6. **Follow-up** — surveillance, PSA monitoring, side-effect management, recurrence detection

## Repository rule

Summaries should help prepare for a clinician discussion, not substitute for it.
